This extension is mainly to install a referral program on your site and share the discount coupons in return.

== Description ==

*Referral marketing (also known as word of mouth marketing) is the oldest but one of the most effective marketing strategies for building a strong customer base and improving eCommerce conversion rates.*

The extension Coupon Referral Program believes the ideology behind referral marketing and lets customers/admin take benefit of the same.

It encourages and enchants more referrals by blending coupons with it (i.e. it promotes the utilization of coupons with referrals). This extension provides discounts coupons to referrers and referred users when they successfully complete the requirement for earning a discount coupon(i.e. users can be rewarded with discount coupons if they signup or refer someone to your e-store).

Customer referral plugin (term can be used as it allows customers to refer their friends and family members) also enables users to share their referral link on social networking platforms and in email inbox. Admin can customize the referral button accordingly using the predefined options or by applying custom CSS.

== Benefits Of Best Referral Coupon Program ==
* Referral reward program builds a strong customer base.
* Elevates engagement on-site, which ultimately improves revenue.
* It generates more loyal customers.
* Enhances Brand Awareness.
* Improves customers’ journey and their lifetime value.
* It provides the ability to share the referral link on Facebook, Twitter, and Email.
* Promotes the practice of referral program.


== Features of Referral Program ==

* **Sign Up Rewards:** The signup discount coupon can be provided to all or only to referred users.
* **Referral Rewards:** The referrer can earn a discount coupon on successful completion of product purchase via referred.
* **Referral Link Sharing:** Coupon Referral Program allows users to share their referral link on Facebook, Twitter, and Email.
* **Customizable Referral Button:** Apply animation and custom CSS to make the referral button look more captivating. It can also be customized using the predefined option present in the Display Configuration
* **Limitation On Coupon Usage:** Individual use of discount coupons can be initiated which means the same coupon can’t be used in conjunction with other coupons. Moreover, the number of times the same coupon can be used can also be fixed.
* **Referral Link Restriction:** The number of days after which the referred can’t get the benefits of being a referred user can be inserted.
* **Maximum orders for referral rewards:** Referrer Reward Limitation- The number of orders upto which the referrer can earn discount coupons on referred purchases can be specified.

* **Referrer Reward Limitation:* The number of orders upto which the referrer can earn discount coupons on referred purchases can be specified.

* **Selection Of Coupon Type:* Admin can specify the Signup/Referral coupon type either set to Fixed/Percentage.

* **Referral Discount Type:* The discount coupon that the referrer will earn on product purchase of referred can be a fixed/percentage value.

* **Percentage Coupon Discount Upto:* Use the “Referral discount amount upto” field to set the max percentage value of the discount coupon. Reward the referrer by sharing percentage amount from the total purchase amount of the referred friend.

* **Discount Amount:* The discount value of coupons for signup and referred users can be specified.

* **Coupon General Settings:* Admin can specify Coupon length and prefix.

* **Set Coupon Expiry Time:* Admin can choose whether he wants to set coupon expiration time or not.

* **Referral Button Visibility:* Referral Button can appear on multiple pages. Admin only needs to decide on which page he wants to display the referral button.

* **Coupon Reporting:* Coupon Reporting can be performed from both the admin and customer end to acquire coupon information. Through Coupon Reporting it’s easy to analyze coupon utilization and its effectiveness for users. Businesses can further use this report to better strategize their promotional strategies.




== Installation ==

The manual installation method involves downloading our Coupon Referral Program and uploading it to your web server via your favorite FTP application. The WordPress codex contains [**instructions on how to do this here**](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).
